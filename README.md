# File transfer with P2P holepunchto/pear library

## Introduction
Main command line interface file for transferring files with Pear (https://github.com/holepunchto/pear) P2P library.

Will accept either -s [file_path] argument for sending the file will emit the hash for transmission (password to be added later)
  or -r [hash] argument, then it will try to download the file seeded by the sending peer
